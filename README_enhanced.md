# Spider – Arduino Stepper Spider Reminder

간단한 목적
------------
출근/퇴근 시간을 잊지 않도록, 정해진 시간대에 "줄에 매달린 거미"가
살아있는 것처럼 오르내리며 사람의 눈에 띄게 만드는 아두이노 기반 알림 장치입니다.

핵심 동작
--------
- 아침: 09:20 ~ 09:40
- 저녁: 18:00 ~ 18:20
- 해당 시간대에만 스텝모터가 랜덤 패턴으로 거미를 위/아래로 움직입니다.

빠른 시작(Quick Start)
----------------------
1. 부품 준비
   - Arduino Uno 또는 Nano
   - 28BYJ-48 스텝모터 + ULN2003 드라이버
   - DS3231 RTC 모듈
   - 점퍼 케이블, 브레드보드, 5V 전원(USB 어댑터 등)
   - 얇은 실(낚싯줄 등)과 가벼운 거미 인형

2. 배선(간단 요약)
   - RTC: SDA → A4, SCL → A5, VCC → 5V, GND → GND
   - ULN2003: IN1 → D8, IN2 → D9, IN3 → D10, IN4 → D11, VCC → 5V, GND → GND
   - 모든 GND는 공통으로 묶습니다.

3. 코드 업로드
   - 사용 가능한 예제 스케치는 `spider_controller/spider_controller.ino`에 있습니다.
   - Arduino IDE 또는 PlatformIO로 스케치를 열어 보드(Arduino Uno/Nano)와 포트를 선택한 뒤 업로드하세요.

4. 첫 실행
   - 시리얼 모니터(9600bps)를 열면 동작 로그를 확인할 수 있습니다.
   - 설치 후 `maxDownSteps` 값을 조정하여 실제 거리가 적절한지 확인하세요.

파일 및 문서 위치
------------------
- 프로젝트 루트: `spider/`
- 주요 문서: `spider_arduino_stepper_project.md` (설계, 회로, 전체 코드 설명)
- 스케치: `spider_controller/spider_controller.ino`

안전 및 권장 사항
------------------
- 모터 전원: Arduino의 5V로 간단히 구동 가능하지만, 안정성(토크/전류)을 위해 별도 5V 전원 어댑터 사용을 권장합니다.
- 한계(리미트) 스위치를 추가하면 과도한 하강/상승을 방지할 수 있습니다.
- 설치 후 거미가 사람이나 물체에 걸리지 않도록 이동 범위를 확인하세요.

문제 해결(간단)
----------------
- 거미가 움직이지 않음: 전원, ULN2003 연결, 그리고 RTC 정상 동작(시간 동기) 확인
- 위치가 틀어짐: `currentSteps`와 `maxDownSteps`를 조정해 보세요
- 이상 소음/전류 부족: 별도 5V 전원 사용 고려
